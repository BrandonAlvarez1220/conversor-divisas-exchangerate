# Conversor de divisas 
Conversor de divisas en java , para el programa oracle ONE , utilizando la API ExchangeRate.

<h2>Divisas de cambio</h2>
<ul>
  <li>MXN</li>
  <li>USD</li>
  <li>EUR</li>
  <li>ARS</li>
  <li>BOB</li>
  <li>BRL</li>
  <li>CLP</li>
  <li>COP</li>
</ul>
